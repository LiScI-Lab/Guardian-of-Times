== Allgemeines

*Guardian of Times (GoT)* ist ein time-tracking tool.
Es ermöglicht die Erfassung von (Arbeits-)zeiten in Teams.
Ursprüngliche Intention hinter GoT ist die automatische Generierung von THM-Stundenzetteln.
